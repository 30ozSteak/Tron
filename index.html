Welcome to Tron! an HTML5 Canvas game based on the classic arcade game of the same name.

This is a two player game where each player controls a lightcycle and create walls around the map as they drive

The first one to crash, be it into a wall, another player, or either players tails, loses the round!

Press Start to Play, Have fun!
